## 🚀 Лучшие проекты

### 🧩 Tic-Tac-Toe Backend (Go)
REST API сервер для игры «Крестики-нолики» с Clean Architecture и DI через [uber/fx](https://github.com/uber-go/fx).  
Показывает владение Go, архитектурой и тестируемым кодом.  
[Репозиторий](https://github.com/ShantaelVeli/s21_tic-tac-toe)

### 🎮 Tetris (C)
Классическая игра Tetris, реализованная с нуля на C.  
Демонстрирует работу с логикой игры, управлением и графикой в консоли.  
[Репозиторий](https://github.com/ShantaelVeli/s21_tetris_shantael)

### 🛠 AptekaPars (C#)
Парсер данных и текстовой информации.  
Хороший пример работы с API, HTML и обработкой данных на C#.  
[Репозиторий](https://github.com/ShantaelVeli/AptekaPars)

### 🐚 Simple Bash Utils (C)
Утилиты, аналогичные `cat` и `grep`, реализованные на C.  
Демонстрирует работу с файловой системой и текстовыми потоками.  
[Репозиторий](https://github.com/ShantaelVeli/s21_simpleBash_shantael)
